Ball Bounce 3D

Game Design Document

Executive Summary:

Game Concept

In this game our main character will be a ball. The game view will be in Third person Player (TPP) mode. This game will have several levels. The higher levels will become more difficult than the previous levels. The main concept of the game is that the ball should be able to move through different type of obstacles in order to pass the level. The ball will have three lives for each level.
The game concept came through the old Nokia (3310) mobile game bounce which was made in 2D in black and white color combination. And was very fun game to play at that time. By keeping that fun in mind, we thought of making a similar game in 2D, but using colorful modern assets that can be fun to play for users and remind them of old days in which they used to play this game.

Genre

The genre of this game will be (Arcade, Casual, Offline).

Target Audience

The Demographic group of the targeted audience will be 6+.

Project Scope:

This is not a huge concept. So, the scope is limited. However, some things can make it broad like adding portals. Furthermore, we can add a story of a square villain who is trying to make every ball shaped thing in the world look like him. So, this game will become more fun to play and to follow the story.

Gameplay:

Objectives

The objective of this game is to target the small age 6+ kids and provide them the old days game fun in the new way. The game has a ball character and different obstacles. There are square shape villains to in the game which will try to affect and stop the ball from passing the particular level. The game should have different levels and the difficulty of the game increases with each level.

Game Progression

There will be different levels in the game. The difficulty of the game increases with each level. The levels have different obstacles and enemies environments try to stop game character to pass the particular level.

In-Game GUI

The game will be in 2D.

Mechanics:

Rules

The ball should be able to move through different type of obstacles in order to pass the level. The ball will have three lives for each level. And the difficulty of other level increases when the user passes a previous level.

Combat

The game has different obstacles and enemies. The player has to dodge them in order to pass the level. The player fails the particular level after losing the three lives.

Physics

The 2D physics will be used on every character and obstacles in game. The ball character can jump, move forward or backward in every direction in order to dodge obstacles.
